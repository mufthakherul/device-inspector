# Archives

This directory holds files that are no longer part of the active project structure but are preserved for historical reference.

## Directory Structure

```
archives/
├── historical/     Sprint summaries, progress snapshots, and planning docs
│                   that captured a point-in-time view during development
└── docs/           Verbose design or implementation write-ups superseded
                    by the canonical docs in /docs
```

## historical/

| File | Description |
|------|-------------|
| `SPRINT_1_SUMMARY.md` | Detailed completion report for Sprint 1 (inventory + SMART + error handling) |
| `SPRINT_2_INFRASTRUCTURE_SUMMARY.md` | Completion report for Sprint 2 infrastructure work (CI/CD, coverage, security) |
| `PROGRESS_SUMMARY.md` | Point-in-time progress snapshot as of 2025-10-28 |
| `IMPLEMENTATION_SUMMARY.md` | Write-up of the v1 CLI packaging implementation |
| `NEXT_STEPS.md` | Sprint 1 era prioritised task list (superseded by ROADMAP.md) |
| `PROJECT_STATUS.md` | Detailed status document as of 2025-10-30 (superseded by ROADMAP.md) |

## docs/

| File | Description |
|------|-------------|
| `V1_PACKAGING_SUMMARY.md` | Verbose packaging solution write-up (superseded by docs/BUILDING.md and docs/DISTRIBUTION.md) |
