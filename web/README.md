# inspecta static report viewer

React + TypeScript static viewer for local `report.json` files.

## Quick start

1. Install dependencies:
   - `npm install`
2. Start viewer:
   - `npm run dev`
3. Open the app and load a local `report.json`

## Scope

- Local, client-side rendering only
- No data leaves your machine
- Shows summary, scores, tests, and raw JSON
